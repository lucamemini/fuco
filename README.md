# fuco


## try it in 4 minutes
```
git clone https://github.com/lucamemini/fuco

# create virtualenv

cd fuco
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
rename fucoconfig.py.template to fucoconfig.py
edit fucoconfig.py with your credentials

./run.sh
```
